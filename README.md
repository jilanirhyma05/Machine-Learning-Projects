# Machine Learning Projects

This repository contains two machine learning projects developed using Python and Google Colab.

## Projects Included

### 1. Heart Disease Prediction
- Predicts whether a person has heart disease
- Uses Logistic Regression
- Dataset includes medical attributes like age, cholesterol, blood pressure

### 2. Diabetes Prediction
- Predicts whether a person is diabetic
- Uses classification techniques
- Dataset includes glucose level, BMI, age, etc.

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Google Colab

## Output
- Both models were trained and tested successfully
- Accuracy achieved:
  - Heart Disease: ~85%
  - Diabetes: ~77%
